# Sistema-de-Gesti-n-de-Solicitudes

## Uso del enum
Se utilizó un enum llamado EstadoSolicitud para representar los estados de una solicitud:
- Pendiente
- EnProceso
- Completada
- Cancelada

Esto permite evitar errores al escribir estados manualmente y mejora la organización del código.

## Funcionalidades
- Registrar solicitudes
- Mostrar solicitudes
- Cambiar estado
- Buscar por ID

## Tecnologías
- C#
- Programación Orientada a Objetos
